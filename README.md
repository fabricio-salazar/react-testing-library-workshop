# react-testing-library-workshop

## Description

React Testing Library 101 - Workshop for Gorilla Logic

## Prerequisites

- Node 16

## How to install the dependencies?

- Run `yarn`

## How to run the app?

- Run `yarn start`

## How to run the tests?

- Run `yarn test`

## How to run a specific test?

- Run `yarn test <test name or path>`
